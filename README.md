## About

### Description and Purpose

A social media website for musicians to
share their tracks. This website is a review of creating
a project with both a frontend and backend from "scratch".

### Inspiration

This website is inspired by slaps and soundcloud,
both of which offer audio streaming functionality.

## Tech Stack

Frontend: React + React Router, Bootstrap, Vite

Backend: Express.js, CloudFlare R2, Render, PostgreSQL
