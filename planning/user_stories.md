# User Stories

## Outline User Roles

1. Musician: a person who creates music and wants to share it

## Draft User Stories

As a [user role], I want to [what], so that [why].

1. As a musician, I want to create an account with email, username, and password so that I can join the music maker community.
2. As a musician, I want to upload an mp3 song as a post with a title and optional description so that others can view my work.
3. As a musician, I want to browse through and search for tracks on the homepage using filters so that I can discover new music.
4. As a musician, I want to play tracks on the homepage so that I can discover new music straight from the website.
5. As a musician, I want to like, comment, and repost tracks created by other musicians so that I can show my support and interests.
6. As a musician, I want to customize my profile picture and bio so that others can see my personality.
7. As a musician, I want to follow other users so that I can make new connections.
8. As a musician, I want to view my followers so that I know who I'm connected with.
9. As a musician, I want to send and receive messages to and from other musicians so that I can make new friends or collaborate with others.
10. As a musician, I want to browse and join communities so that I can connect with like-minded individuals.
11. As a musician, I want to view notifications for interactions (such as likes, comments, reposts, or messages) so that I can see who's interacting with me.
12. As a musician, I want to create lists of songs uploaded by other musicians so that I can be inspired and keep track of what I like.
13. As a musician, I want to edit my mp3 song post's title or description so that I can change my mind about what I create.
14. As a musician, I want to delete posts whenever I want to so that avoid sharing a song I regret creating.
15. As a musician, I want to view the detail page for a posted track so that I can see its comments and full details.
